- (decision) Paradigm: Ports & Adapters (hexagonal-lite) backend. routers->services->repository ports; DynamoDB access ONLY inside repository adapters. Prevents boto3/logic leaking into HTTP layer & scattered data access.
- (decision) Guest identity: opaque server-issued guestId (UUID) via X-Guest-Token header, stored client-side (localStorage). Carts/Orders keyed by guestId. No auth, no user records. Prevents divergent session mechanisms & PII.
- (decision) Persistence: separate DynamoDB tables per aggregate (Products, Carts, Orders); each owned by one repository. Multi-table for POC clarity; single-table design noted as deferred alternative.
- (decision) Catalog search (KEY DECISION - Option A, needs user confirm): real DynamoDB GSIs. gsi_category(PK=category,SK=price) for facet+price sort+pagination; gsi_listing(PK=const,SK=price) for unfiltered list. Free-text via FilterExpression contains(name/description). Cursor pagination via LastEvaluatedKey. Limitation: no native full-text (OpenSearch deferred). Alternative B: in-memory catalog cache.
- (decision) Money: integer minor units (cents), single currency. Subtotal=sum(unitPrice*qty); OrderTotal=Subtotal+flat shipping const (config); no tax. Prevents float drift & divergent total math.
- (decision) Order immutability: placing order writes immutable Order snapshot (lines+totals+shipping+ref+timestamp) AND clears Cart in one operation; Orders never updated after creation.
- (decision) API contract: REST/JSON; FastAPI OpenAPI is source of truth; error envelope {error:{code,message}}; JSON camelCase; opaque base64 pagination cursor; frontend consumes a typed client derived from OpenAPI.
- (decision) Config/env parity (12-factor): DynamoDB endpoint, table names, shipping const, CORS via env vars. Only local-vs-AWS diff is DynamoDB endpoint+creds. API stateless (all mutable state in DynamoDB by guestId).
- (version) Verified 2026-07: Python 3.13, FastAPI 0.139.x, Uvicorn, Pydantic v2, boto3; Node 22 LTS, TypeScript 5.x, React 19.2, Vite 8.1; amazon/dynamodb-local Docker; Docker Compose.
- (question) AWS ECS/Fargate deployment + IaC deferred (enhancement). Local orchestration via docker-compose (api + dynamodb-local + frontend) is the v1 operational envelope.
